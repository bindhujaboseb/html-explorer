📖 Topic A — What is HTML, Editors & Basic Structure

🌐 What is HTML?

HTML stands for HyperText Markup Language. It is the skeleton of every website you visit! 🦴

Think of it like a recipe 🍳 — HTML tells the browser what to show on the page, just like a recipe tells a chef what to cook!

🇦🇪 UAE Connect: The website for the Burj Khalifa (burjkhalifa.ae) was built using HTML, CSS and JavaScript — just like you'll learn today!

When you type a web address like www.google.com, your browser reads HTML code and shows you a beautiful page. Cool, right? 😎

🛠️ HTML Editors

To write HTML, you need a text editor — it's like your coding notebook!

📝 Notepad
Simple editor built into Windows. Free & easy!
💙 VS Code
Popular professional editor with syntax colours!
🌐 Notepad++
Better than Notepad with helpful features.
☁️ CodePen
Online editor — no install needed!
💡 Tip: Save your file with .html extension (e.g. mypage.html) then open in Chrome!

🏗️ Basic HTML Structure

Every HTML page has the same basic skeleton — like the frame of the Burj Khalifa! 🗼

<!-- Every HTML page starts like this --> <html> <!-- The whole website --> <head> <!-- Brain of the page (not shown) --> <title>My Page</title> <!-- Tab title --> </head> <body> <!-- What users SEE --> <h1>Hello World!</h1> <p>I love UAE! 🇦🇪</p> </body> </html>
  • 🔵 <html> — The container for everything
  • 🟡 <head> — Hidden info (title, settings)
  • 🟢 <body> — Everything you see on screen!

📖 Topic B — Important HTML Tags & Elements

📋 Important HTML Tags

<h1>–<h3>
Headings — h1 biggest, h3 smallest. Like titles!
<p>
Paragraph — for writing sentences and text.
<img>
Image — shows a picture. No closing tag needed!
<a>
Anchor — creates a clickable hyperlink.
<br>
Line Break — moves to next line. Like pressing Enter!
<hr>
Horizontal Rule — draws a line across the page.
<ul> <li>
Unordered list — bullet points!
<ol> <li>
Ordered list — numbered (1, 2, 3...).
<div>
Division — a box to group elements together.
<span>
Inline container — for styling part of a sentence.
<table>
Table — displays data in rows and columns.
<form>
Form — collects user input (name, email etc.).

🏷️ HTML Tags & Elements

An HTML tag is like a label. Most tags come in pairs — an opening tag and a closing tag:

<h1>Welcome to UAE!</h1> ↑ Opening tag ↑ Content ↑ Closing tag (has /)

Opening + content + closing = one HTML Element 🎁

Some tags are self-closing — they don't need a closing tag:

<br> <!-- line break — no closing tag --> <hr> <!-- horizontal line — no closing tag --> <img> <!-- image — also self-closing! -->

📖 Topic C — Text Formatting Tags & Attributes

✍️ HTML Text Formatting Tags

<b>
Bold text — makes text thick and heavy.
<i>
Italic text — makes text lean sideways.
<u>
Underline — draws a line under text.
<strong>
Strong — like bold but means "important".
<em>
Emphasize — like italic but means "emphasis".
<mark>
Highlight — yellow highlight on text.
<sup>
Superscript — text above the line (2²).
<sub>
Subscript — text below the line (H₂O).
<del>
Deleted — strikethrough text.
<small>
Small text — makes text smaller.

🎛️ HTML Attributes

Attributes give extra information to HTML tags. They go inside the opening tag:

<img src="burj.jpg" alt="Burj Khalifa 🗼" width="200"> <a href="https://www.uae.gov.ae">UAE Website</a> <p style="color:blue;">Blue text!</p>
  • 🔗 href — the link address for <a> tags
  • 🖼️ src — the image file path for <img> tags
  • 📝 alt — description of an image (for accessibility)
  • 📐 width / height — size of images
  • 🎨 style — add CSS styling directly
  • 🆔 id / class — names for styling or scripting
